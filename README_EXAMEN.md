He estructurado el proyecto separando la lógica en módulos ES6 para mantener el código limpio y organizado. En el `index.html` cargo el script principal `main.js` como módulo, que se encarga de inicializar la aplicación cuando salta el evento `DOMContentLoaded`. La gestión del formulario está en `formController.js`, donde uso eventos `input` para validaciones en tiempo real y `submit` para procesar los datos, utilizando siempre `preventDefault()` para evitar que la página se recargue y perder el estado.

Para asegurar la integridad de los datos, he centralizado las validaciones en `validators.js`, apoyándome en `regex.js` para las expresiones regulares de nombres y resultados. También compruebo que la fecha no sea futura y que los jugadores no sean la misma persona. La interfaz se actualiza desde `ui.js`, separando la vista de la lógica, y guardo las partidas en el array de `data.js`. Con esta estructura modular, el proyecto es escalable y fácil de mantener, cumpliendo con todos los requisitos del examen.
